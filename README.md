# Linux Log Monitoring Automation

A simple DevOps automation project using Bash and Cron to monitor Linux system logs and detect critical issues automatically.

## Features
- Detects error, failed, warning, and critical logs
- Generates alert reports
- Automated scheduling using cron jobs
- Helps in basic operational monitoring

## Tech Stack
- Bash
- Linux
- Cron

## How to Run

```bash
chmod +x log_monitor.sh
./log_monitor.sh
